# How to install

## Pre-requisites

* `sfdx-cli`

## How to install

Inside your sfdx project of choice, open a terminal and run

```sh
sfdx plugins:install sfdocs-sfdx-plugin
```

### Verify the installation

Run the context help of the plugin

```sh
sfdx sfdocs --help
```

or check in the list of installed plugins

```sh
sfdx plugins
```
